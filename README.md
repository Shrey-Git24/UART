# UART

 Developed a complete end-to-end simulation of the UART (Universal Asynchronous Receiver-Transmitter) proto-
col, including frame assembly, BPSK modulation, and AWGN channel modeling.
– Implemented Even Parity error detection and frame validation logic (Start/Stop bit verification) to simulate real-
world hardware peripheral behavior.
– Engineered a Monte Carlo simulation to analyze the relationship between Signal-to-Noise Ratio (SNR) and Bit
Error Rate (BER) across 103 iterations.
– Evaluated protocol efficiency by calculating the Successful Frame Ratio (Throughput), identifying critical SNR
thresholds for reliable data transmission in noisy environments.
– Implemented receiver-side frame synchronization and parity checking for error detection and successful packet
validation.
